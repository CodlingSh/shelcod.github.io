Morse code buzzer

View on github

A super simple, yet super fun little Python project I made to play around with the GPIO pins on a Raspberry Pi. The program runs in a terminal and asks the user to input a string, and then converts that string to the proper morse code characters. Then the program will display the morse code characters in the terminal and either flash an LED or buzz the piezo buzzer (or both) in the correct morse code pattern at approximatley 20 words per second.
